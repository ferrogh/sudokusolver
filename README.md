# sudokusolver

## Installation instructions

1. Installing minisat

    * Using Linux or WSL:

        Clone the repository
        ```
        git clone https://github.com/niklasso/minisat.git
        ```

        Open the directory
        ```
        cd minisat/
        ```

        Make the files
        ```
        sudo make install
        ```

        If you get error while loading shared libraries
        ```
        sudo /sbin/ldconfig -v
        ```
